# Mon Blog Professionnel

Bienvenue sur mon blog professionnel, où je partage mon parcours, mes expériences et mes compétences en informatique, ainsi que mes projets réalisés durant mon stage. 

Vous pouvez explorer les différentes sections du site pour en savoir plus sur moi, consulter mon CV, lire mes articles, ou me contacter pour toute question ou opportunité.

## 🔗 Lien du projet

Vous pouvez visiter le site en ligne ici : [Mon Blog Professionnel](https://sashabrun.github.io/Stage/)

## 📁 Structure du projet

Le projet est structuré en plusieurs fichiers et dossiers :

- **index.html** : Page d'animation du site.
- **home.html** : Page d'accueil du site.
- **contact.html** : Page pour me contacter et en savoir plus sur l'entreprise de mon stage.
- **articles.html** : Page listant les articles détaillant les différentes expériences et projets réalisés durant mon stage.
- **Front/css/** : Contient les fichiers CSS pour styliser les pages web.
- **img/** : Contient les images utilisées sur le site, telles que mon CV, les images des projets, etc.

## 🛠️ Technologies utilisées

- **HTML5** : Pour la structure des pages web.
- **CSS3** : Pour le style et la mise en page des pages web.
- **GitHub Pages** : Pour héberger le site en ligne.

## 🚀 Déploiement

Le site est hébergé sur GitHub Pages. Pour déployer les modifications :

1. Poussez vos changements sur la branche principale du dépôt GitHub.
2. GitHub Pages se chargera automatiquement de déployer la dernière version du site.
